let

We all know var, kinda? But what is and how does 'let' work?

'let' is a more block related, a bit more temp kind of a variable that only live in its declared

Let Example


  // Let example
  function f() {
    let x = 99;

      let x = 3; // other variable

    }
    console.log(x); // 99
  }

Const

Works in similiar fashion as other languages with the same type. It doesn't like getting overwritten! Main advantage is the reader of the code knows that this variable shouldn't get overwritten.

Const example


  // Assigning to A const variable is a syntax error
  const A = 1; A = 2;

  // const requires an initializer
  const FOO; // SyntaxError: missing = in const declaration

  // const also works on objects
  const MY_OBJECT = {"key": "value"};

  // Overwriting the object fails as above (in Firefox and Chrome but not in Safari)
  MY_OBJECT = {"OTHER_KEY": "value"};

  // However, object keys are not protected,
  // so the following statement is executed without problems
  MY_OBJECT.key = "otherValue"; // Use Object.freeze() to make object immutable
